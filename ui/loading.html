Loading 组件

  1. 语法
    ui.Loading(options)
  2. 文档
    参数名称 类型 | 是否必填 默认值 释义
    className String | false '' 根组件元素的className
    html String | false '' loading 元素的html代码片段
    parentNode Element | false document.body 承载该组件的父元素
    start Function | false Function loading开始
    end Function | false Function loading结束

  3. demo

    js核心代码

                
              var startLoadingBtn = document.getElementById('start_loading_btn')
              var endLoadingBtn = document.getElementById('end_loading_btn')
              startLoadingBtn.addEventListener('click', function () {
                var loading = ui.Loading().start()
                setTimeout(function () {loading.end()}, 5000)
              })
