使用方法

  1. 使用npm import方式引入(Dialog为例)

    npm i yao-m-ui --save
    import * as ui form 'yao-m-ui'
    import "yao-m-ui/widget.css""
    ui.Dialog({})

  2. 或者使用require加载

    npm i yao-m-ui --save
    var ui = require('yao-m-ui')
    require("yao-m-ui/widget.css"")
    ui.Dialog({})

  3. 或者直接script 标签引入 npm包目录下的index.js

    <script src='./index.js></script> window.YAO_M_UI.Dialog({})
