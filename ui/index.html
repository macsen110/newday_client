使用方法

  1. 使用npm import方式引入(Dialog为例)

    npm i yao-m-ui --save
    import * as ui form 'yao-m-ui'
    import "yao-m-ui/widget.css""
    ui.Dialog({})

  2. 或者使用require加载

    npm i yao-m-ui --save
    var ui = require('yao-m-ui')
    require("yao-m-ui/widget.css"")
    ui.Dialog({})

  3. 或者直接script 标签引入 npm包目录下的index.js

    <script src='./index.js></script> window.yaoMUI.Dialog({})

Tab 切换组件

面向项目设计的选项卡，对类名和层级等无依赖，大家在布局的时候可以很灵活。支持异步添加的tab nav选项
  1. 语法
    ui.TabWidget(options)
  2. 文档
    参数名称 类型 | 是否必填 默认值 释义
    curIdex Number | false 0 选中选项卡的索引值。默认0
    activeClass String | false .on 当前选中nav标签和content标签的激活class 类名
    tabNavContainer String | false .tab-nav-container tab 导航标签的父元素
    tabConContainer String | false .tab-con-container tab 内容标签的父元素
    tabNavItems LikeArray | false 为空 tab 导航标签
    例如 document.querySelectorAll('.nav-items'),
    不写的话默认取tabNavContainer的子元素
    tabConItems LikeArray | false 为空 tab 内容标签
    例如 document.querySelectorAll('.nav-con-items')
    不写的话默认取tabConContainer子元素
    tabConItems 与 tabConContainer必填一项
    callback Function undefined 点击切换的回调。其中，上下文this指点击元素DOM对象。默认参数为当前点击的索引值
  3. demo 
                        ui.TabWidget({
                      curIdex: 1,
                      callback: function (idx) {
                        alert(idx)
                      }
                    })
                  
                          body {
                        color:red;
                        cb: fuc
                      }
