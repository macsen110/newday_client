Tab 切换组件

面向项目设计的选项卡, 支持异步添加的tab nav选项
  1. 语法
    ui.TabWidget(options)
  2. 文档
    参数名称 类型 | 是否必填 默认值 释义
    curIdex Number | false 0 选中选项卡的索引值。默认 0
    activeClass String | false on 当前选中nav标签和content标签的激活class 类名
    tabNavContainer Object | false document.querySelector('.tab-nav-container') tab 导航标签的父元素
    tabConContainer Object | false document.querySelector('.tab-con-container') tab 内容标签的父元素
    tabNavItems LikeArray | false 为空 tab 导航标签
    例如 document.querySelectorAll('.nav-items'),
    不写的话默认取tabNavContainer的子元素
    tabConItems LikeArray | false 为空 tab 内容标签
    例如 document.querySelectorAll('.nav-con-items')
    不写的话默认取tabConContainer子元素
    callback Function undefined 点击切换的回调。其中，上下文 this指点击元素DOM对象。默认参数为当前点击的索引值

  3. demo,点击add 按钮可以增加nav标签

    html 代码片段

    
              <ul class="tab-nav-container">
                <li class="item">nav-1</li>
                <li class="item">nav-2</li>
              </ul>
              <ul class="tab-con-container">
                <li class="item">con-1</li>
                <li class="item">con-2</li>
              </ul>

    js核心代码

                
              ui.TabWidget({
                curIdex: 1,
                callback: function (idx) {
                  console.log(idx)
                }
              })
