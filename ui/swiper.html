easyMove 滑动插件

简单的滑动插件,可用于pc和h5内置集成了swiper,简单易用,功能强悍
  1. 语法
    ui.easyMove(element, options)
  2. element 滚动目标dom 节点
  3. options 文档解析
    参数名称 类型 | 是否必填 默认值 释义
    parentEle String | false '' 手指touch或鼠标移动dom块 默认为 element
    index Number | false 0 move 定位到哪个子item
    focusIndex Number | false 0 聚焦到某个位置的索引值
    speed Number | false '' 滚动速率
    limitBorder Bollen | false true 滑动后是否会回顶到边界
    showNum Number | true 显示的滑动的item数量
    distance Number | false 50 自定义滑动距离,超过这个距离到达下一个场景
    orientation Number | false 1 滑动方向 1为横向,2为纵向
    touchMoveCb Function | false Function 监听动画滑动事件回调
    callback Function | false Function 动画完成后的回调
    autoPlay Bollen | false false 是否可以自动播放
    paginationList Element | Object null 指示导航,多用于swiper滚动时下面的指示灯

  4. 类swiper滑动

    html核心代码

                  
                <ul class="easy-move-list">
                    <li class="item">01</li>
                    <li class="item">02</li>
                    <li class="item" data-index=2>03</li>
                </ul>
                <ul class="pagination-list">
                  <li class="item"></li>
                  <li class="item"></li>
                  <li class="item"></li>
                </ul>

    js核心代码

                  
                  ui.easyMove(document.querySelector('.easy-move-list'), {
                    autoPlay: true,
                    paginationList: document.querySelector('.pagination-list')
                  })

                    showNum:4,
                    limitBorder: true,
                    touchMoveCb: function () {
                      touchMoveCb.bind(easyMove)(easyMove.index)
                    },
                    callback: function (params) {
                      easyCallback.bind(easyMove)(easyMove.index)
