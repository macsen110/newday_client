Dialog 弹框插件

面向项目设计的弹框插件, 简单易用,配置可控
注意点

1.关闭事件按钮元素类名需包含icon-dialog-cancel

2.取消事件按钮元素类名需包含btn-dialog-cancel

3.确定事件按钮元素类名须包括btn-dialog-ok

4.遮罩层元素类名须包括dialog-mask-bg

  1. 语法
    ui.Dialog(options)
  2. 文档
    参数名称 类型 | 是否必填 默认值 释义
    className String | false '' 组件root 标签下传入的css class名称, 业务中定制化组件需要在此重写
    title String | false '' 弹框的标题值可以为字符串或者html片段
    content String | false '' 弹框的主体内容可以为字符串或者html代码片段
    foot String | false '' 弹框的尾部内容,一般是按钮组合,比如关闭取消按钮

    <button class="btn-dialog-cancel">取消</button> <button class="btn-dialog-ok">确定</button>

    afterOpen Function | false Function Dilog代码片段render到body中以后就调用,弹框内若有元素有绑定事件应写在此函数内
    afterClose Function | false Function 点击关闭(取消按钮)之后触发的事件
    afterOk Function | false Function 点击确定按钮之后触发的事件
    canMaskClose Boolen | false true 点击蒙层是否调用关闭弹框事件
    isMask Bollen | false true 弹框组件是否需要蒙层
    maskAction Function | false undefined 为点击蒙层而准备的函数
    close Function | false Function 点击取消按钮(btn-dialog-cancel), 关闭按钮(btn-dialog-close)触发
    destory Function | false Function 直接在页面中移除弹框

  3. js核心代码

                
                ui.Dialog({
                  className: 'some-dialog-class'
                  title: '我是标题',
                  content: '我是内容',          
                  afterClose: function() {
                    console.log('我是点击关闭按钮以后调用的')
                  },
                  afterOpen: function () {
                    console.log('我是弹框触渲染到页面发以后弹的')
                  },
                  afterOk: function () {
                    console.log('我是点击确定按钮触发的')
                  },
                  foot: '<button class="btn-dialog-ok">确定</button><button class="btn-dialog-cancel">取消<button>'
                })
